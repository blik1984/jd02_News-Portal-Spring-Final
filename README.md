# News Portal Web Application (Spring Boot)
# Enterprise-level web application developed as part of Java Enterprise training.
# Implemented user registration, authentication, role-based access control, news management, and comments.
# Key features:
# • User registration and authentication (Spring Security, BCrypt)
# • Role-based authorization (ADMIN / USER / GUEST)
# • CRUD operations for users, news, comments
# • Pagination and filtering
# • MVC architecture (Controller / Service / DAO)
# • Hibernate ORM with HQL
# • Transaction management
# • Logging and exception handling
# Technologies: Java, Spring Boot, Spring MVC, Spring Security, Hibernate, SQL, Thymeleaf, Maven
